EMAIL Supabase Authentication Emails
Provider: Supabase Auth (managed)
Customization: Via Supabase dashboard email templates

Three auth emails are handled by Supabase:

  1. Sign Up Confirmation — Sent when user creates account
    Contains email verification link; user must click to confirm account
  2. Password Reset — Sent when user requests password reset via forgot-password page
    Contains reset link; configurable redirect URL (sent to tenant-specific subdomain)
  3. Email Change Confirmation — Sent when user updates account email in settings
    Contains verification link for new email address
📧 Configuration: Supabase email templates are customized in the Supabase project dashboard under Authentication > Email Templates. These can be branded with your logo and messaging.
🔄 Redirect URLs: Password reset link redirects to https://[tenant-slug].flowlabsolutions.au/reset-password in production or http://[tenant-slug].localhost:3001/reset-password in development.

🎨 Email Branding System

All emails are automatically branded with tenant settings:

  • Logo: Tenant's custom logo URL or business name fallback
  • Primary Colour: Tenant's primary brand colour (header background), defaults to #3B82F6
  • Business Name: Auto-populated from tenant profile
  • Contact Details: Phone and email from tenant profile footer
✓ Responsive Design: All templates use mobile-first HTML tables for reliable rendering across email clients (Gmail, Outlook, Apple Mail, mobile).
