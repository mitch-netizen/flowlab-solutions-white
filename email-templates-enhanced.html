📡 Integration & CTA Summary

Link Types & Portal Routes

Customer Public Links (unauthenticated)

Action URL Pattern Authentication
View booking flowlab.app/jobs/[token] Public token in URL
Pay invoice flowlab.app/invoice/[token]/pay Public token in URL
Leave feedback flowlab.app/feedback/[token] Public token in URL
Accept quote flowlab.app/quote/[token]/accept Public token in URL

Operator Dashboard Links (authenticated)

Action URL Pattern Notes
View job [tenant].flowlabsolutions.au/dashboard/jobs/[id] Full job details, status updates
View invoice [tenant].flowlabsolutions.au/dashboard/invoices/[id] Payment status, sync history
View customer [tenant].flowlabsolutions.au/dashboard/crm/customers/[id] Contact info, job history
Morning digest [tenant].flowlabsolutions.au/dashboard Main dashboard overview
Automation health [tenant].flowlabsolutions.au/dashboard/system-health Error logs, dead letters
Xero integration [tenant].flowlabsolutions.au/dashboard/integrations/xero Config, sync status, auth
🔗 Dynamic URLs: Replace [tenant] with tenant slug (e.g., plumbing-co) and [id]/[token] with actual IDs/tokens from database.
Integration Checklist

✅ Customer Notifications

  • ✓ Booking confirmation → View portal link
  • ✓ Job complete → Feedback form link + public portal
  • ✓ Invoice issued → Payment link (Stripe/gateway)
  • ✓ Payment reminder → Payment link

✅ Operator Notifications

  • ✓ New enquiry → View in CRM + Quick quote action
  • ✓ Morning digest → Scheduler link, invoice dashboard
  • ✓ Automation failure → Logs, config, retry button
  • ✓ Payment received → Dashboard invoice view

✅ Xero Integration (MCP)

  • ✓ Invoice synced → View in Xero + FlowLab
  • ✓ Payment received → Auto-mark paid (webhook)
  • ✓ Sync error → Debug links, retry action
  • ✓ Daily sync report → Sync status dashboard

✅ Supabase Auth (MCP)

  • ✓ Email verification → Supabase managed
  • ✓ Password reset → Tenant-specific redirect
  • ✓ Email change → Supabase managed
  • ✓ Session management → Supabase dashboard
